发表于 2023-07-09 Valine：

导数复习

常用技巧

放缩

T1

题面

$f(x) = 2x-\sin x-\sqrt{a}\ln x$

若存在 $x_1,x_2(0<x_1<x_2)$，使 $f(x_1) = f(x_2)$，求证 $x_1x_2<a$。

解析

初步判断

第一眼看上去，$f(x)$ 里面竟然有三个基本函数！这直接硬导肯定没有出路。

再看证明的是一个不等式，大概率就要用基本函数之间的关系放缩了。

那么，我们就来想一想，这些函数之间的关系有哪些？

工具

在 $x>0$ 时，有：

$x>\sin x$

$\ln x\leq x-1$

尝试解答

$2x_1-\sin x_1-\sqrt{a}\ln x_1 = 2x_2-\sin x_2-\sqrt{a}\ln x_2$

这里有一个技巧，因为我们要求的是关于 $a$ 的，所以把带 $a$ 的放一边。

$2(x_1-x_2)-(\sin x_1-\sin x_2) = \sqrt{a}(\ln x_1-\ln x_2)$

先拿 $x>\sin x$ 放缩，就有 $\sin x_1-\sin x_2<x_1-x_2$，带回原式。

$x_1-x_2<\sqrt{a}(\ln x_1-\ln x_2)$

$\sqrt{a}>\frac{x_1-x_2}{\ln x_1-\ln x_2}$

到这一步，很多同学肯定已经有了答案，这里还有一个隐含的，但是又非常常用的放缩没有提及：

$\frac{x_1-x_2}{\ln x_1-\ln x_2}>\sqrt{x_1x_2}$

补充知识

众所周知，我们刚进高一时就学到了这样一个不等式：

$\frac{2}{\frac{1}{a}+\frac{1}{b}}\leq \sqrt{ab}\leq \frac{a+b}{2}\leq \sqrt{\frac{a^2+b^2}{2}}$）（$a = b$ 时取等）

现在，升到了高三，就必须知道对数不等式了：

$\sqrt{ab}<\frac{a-b}{\ln a-\ln b}<\frac{a+b}{2}$

证明网上有很多，自己找着看，不难，里面的换元思想很重要。

最终解答

知道了上面的不等式，就可以很简单地推出：

$\sqrt{a}>\sqrt{x_1x_2}$

即 $x_1x_2<a$。

总结

看到多个基本函数时不要慌，一个个放缩，化成一样的。

一些常用的不等式要记得，有时候真的是棋差一着。

这道题其实不难，对吧？

T2

题面

当 $x>2$ 时，$x\ln(x-1)<a(x-2)$ 有解，求 $a$ 的取值范围。

解析

初步判断

两边的函数都很简单，并且只有两个初等函数，参变分离之后求导不难（而且题目里面的 $x>2$ 相当于提示你把它除过去），考虑先参变分离，再求出含 $x$ 的函数的最小值。

工具

$\ln x\leq x-1$

尝试解答

$a>\frac{x\ln (x-1)}{x-2} = f(x)$

$f^{‘}(x) = \frac{\frac{x^2-2x}{x-1}-2\ln (x-1)}{(x-2)^2}$

底下的东西和正负无关，去除。

$\frac{x^2-2x}{x-1}-2\ln(x-1)$

$(x-1)-\frac{1}{x-1}-2\ln(x-1)$

一个飘带一个对数，都是单增的，不好判断正负，考虑放缩：

$(x-1)-\frac{1}{x-1}-2\ln(x-1)>(x-1)-\frac{1}{x-1}-2x+4 = -x-\frac{1}{x-1}+3$

可惜，右边的在 $x$ 极大时为负，判断不了原式正负，放太大了。

再次观察

$f^{‘}(2) = 0$

这也是为什么不能放缩的原因，$x$ 取的非常近，容易有偏差。在不能放缩，且导函数值十分精确的情况下，考虑求二阶导。

$f^{‘’}(x) = 1+\frac{1}{(x-1)^2}-\frac{2}{x-1}$

这不就是个二次函数吗，对称轴：$\frac{1}{x-1} = 1\rightarrow x = 2$。

$x>2$ 时$f^{‘’}(x)>0$。

于是，$f^{‘}(x)$ 单增，$x>2$ 时 $f^{‘}(x)>0$，$f(x)$ 在 $x = 2$ 时取最小值。

但是，我们还面临一个难题，$x = 2$ 时，$f(x)$ 没有意义。

扩展知识

我们发现，要求的实际上就是 $\lim_{x\rightarrow 2}\frac{x\ln (x-1)}{x-2}$。

对于这种分子分母都是 $0$，或者都是正无穷的极限，就要用洛必达法则了。

洛必达法则：$\lim\frac{f(x)}{g(x)} = \lim\frac{f^{‘}(x)}{g^{‘}(x)}$。

解法1

开洛！开导！

$\lim_{x\rightarrow 2}\frac{x\ln (x-1)}{x-2} = \lim_{x\rightarrow 2}\frac{\ln(x-1)+\frac{x}{x-1}}{1}$

把 $x = 2$ 带进去，可以直接得到极限为 $2$。

解法2

看到 $\ln$，第一个想到的应该是放缩。

但是刚刚放一阶导不是失败了吗，为什么又要放呢？

因为这题的条件十分特殊，我们要求的恰好是 $x = 2$ 时的极限。

放缩，就是 $\ln x\leq x-1$，而这个等号从何而来？就是在 $x = 1$ 时取等！

所以，当 $x = 2$ 时，$\ln(x-1) = x-2$！

于是，原式可化为 $\frac{x(x-2)}{x-2} = x$。

显然极限为 $2$。

总结

要弄明白问题的实质，这里的放缩，有成有败，实际上是有原因的，自己要体会。

洛必达法则虽然好用，但是解答题慎用，除非实在做不出来才用。

T3

题面

存在 $a\in R$，使得对任意 $x\in [\frac{1}{e},e]$，不等式 $\ln x\leq ax^2+bx\leq (e^2-2e)\ln x+e$ 恒成立，求 $b$ 的最小值。

解析

初步判断

一般的双变量问题都是先定一个变量，之后再确定另一个的范围。

但是，这道题中，$a$ 没有任何范围，我们不好处理。

不过，中间的东西比较简洁，可以看成一个二次函数，但是这样不好让它在两个函数之间。

考虑同时除 $x$，把中间的东西变成一个直线，就可以方便的放进去了。

尝试解答

$f(x) = \frac{\ln x}{x}$

$f^{‘}(x) = \frac{1-\ln x}{x^2}$

$g(x) = \frac{(e^2-2e)\ln x+e}{x}$

$g^{‘}(x) = \frac{e^2-2e-(e^2-2e)\ln x-e}{x^2} = \frac{(2e-e^2)\ln x+e^2-3e}{x^2}$

截距的最小值就是 $P$ 作函数 $f(x)$ 的切线，这样才能保证这条直线在两个函数之间。

设切点为 $(x_0,\frac{\ln x_0}{x_0})$。

对于斜率，我们可以用两点式和导数分别算出，而且它们是相等的：

$\frac{e-1-\frac{\ln x_0}{x_0}}{e-x_0} = \frac{1-\ln x_0}{x_0^2}$

解得 $x_0 = 1$。

于是可以算出斜率等于 $\frac{1-0}{1^2} = 1$，进而求出 $b = -1$。

总结

这道题的主要难点在于想到用直线代替二次函数。

T4

题面

已知直线 $y = kx+b$ 和函数 $y = e^x$ 相切于点 $(x_1,y_1)$，和函数 $y = \ln x$ 相切于点 $(x_2,y_2)$，若 $x_2>1$，且 $x_2\in (n,n+1)$，$n \in \Z$，求 $n$。

解析

初步判断

乍一看，好像有 $4$ 个变量，而且我们还真能列出 $4$ 个方程，但是，这样就会陷入繁杂的计算中，且最后得到的式子不会很优美（需要多次放缩）。

不过，仔细分析一下，会发现 $y = kx+b$ 不就是公切线吗？那我们其实完全不用管 $k$ 和 $b$，它们被两个切点唯一确定了，于是只用找出两个切点之间的关系了。

尝试解答

我们可以把过两个点的切线分别写出来：

$y = e^{x_1}x+e^{x_1}-e^{x_1}x_1$

$y = \frac{1}{x_2}x+\ln x_2-1$

由于是公切线，所以这两条直线的斜率和截距是相等的。

$e^{x_1} = \frac{1}{x_2}$

$e^{x_1}-e^{x_1}x_1 = \ln x_2-1$

$\frac{1}{x_2}(1+\ln x_2) = \ln x_2-1$

$x_2\ln x_2-x_2-\ln x_2-1 = 0$

有了这个方程，我们是肯定可以把 $x_2$ 解出来的，但是在高中阶段不是很可解，这也是为什么题目让我们求 $x_2$ 所处的区间。

$f(x) = x\ln x-x-\ln x-1$

$f^{‘}(x) = \ln x+1-1-\frac{1}{x} = \ln x-\frac{1}{x}$

在 $x > 2$ 时，$f^{‘}(x)>0$，且 $f(2) = \ln 2-3<0$。

$f(3)<0,f(4)<0,f(5)>0$，所以 $n = 4$。

总结

这道题比较迷惑人的一点在于给出了毫无用处的 $y = kx+b$，所以今后做题一定要明确题目中各个变量之间的关系，尽量减少变量个数，以起到简化计算的效果。

T5

题面

已知 $f(x) = ae^x-x^2$ 有两个极值点 $x_1,x_2$。若 $x_2\geq 3x_1$ 时，$x_1+\lambda x_2\geq 2x_1x_2$ 恒成立，求 $\lambda$ 的最小值。

解析

初步判断

这种含有两个变量的题目叫做双变量问题，常见的处理方法有极值点偏移、单变量表示。这里既然给出了 $x_1,x_2$ 之间的倍数关系，就可以设 $t = \frac{x_1}{x_2}$，这样即可以不用 $a$ 就把两个变量表示出来，也可以知道 $t$ 的范围。

尝试解答

令 $x_1 = tx_2$，则 $t\leq \frac{1}{3}$。

下面的几步是常见的处理，主要目的是消掉 $a$，把 $x_1$ 代换为 $tx_2$，把 $t$ 看成已知量，解出用 $t$ 表示 $x_2$ 的式子，请第一次接触的读者深入理解：

$ae^{x_1} = 2x_1$

$ae^{x_2} = 2x_2$

相除，消掉 $a$：

$\frac{e^{x_1}}{e^{x_2}} = \frac{x_1}{x_2}$

带入 $x_1 = tx_2$：

$\frac{e^{tx_2}}{e^{x_2}} = t$

解出 $x_2$：

$e^{tx_2} = te^{x_2}$

$tx_2 = \ln t+x_2$

$x_2 = \frac{\ln t}{t-1}$

$x_1 = tx_2 = \frac{t\ln t}{t-1}$

（没看懂的再看一遍，之后用这种方法解题，都会直接解出 $x_1,x_2$，不会再写详细过程了）

带入题面中不等式：

$\frac{t\ln t}{t-1}+\lambda \frac{\ln t}{t-1}\geq 2\frac{t\ln^2t}{(t-1)^2}$

$\lambda\geq \frac{2t\ln t}{t-1}-t = f(t)$

$f^{‘}(t) = \frac{2(\ln t+1)(t-1)-2t\ln t}{(t-1)^2}-1 = \frac{2(t-\ln t-1)}{(t-1)^2}-1$

当 $f^{‘}(t)>0$ 时，$2(t-\ln t-1)\geq (t-1)^2$，即 $t^2-4t+3+2\ln t<0$。

这个东西不好直接看，再求导：

$g^{‘}(t) = 2t-4+\frac{2}{t}\geq 2\sqrt{2t\frac{2}{t}}-4 = 0$

所以前面的 $t^2-4t+3+2\ln t$ 单增，又发现它在 $t = 1$ 时等于 $0$，于是在 $t\in (0,1)$ 时小于 $0$，这覆盖了 $t$ 的范围。

所以推出 $f^{‘}(t)$ 在 $(0,\frac{1}{3})$ 内恒大于 $0$，即 $f(t)$ 单增。

所以 $\lambda$ 大于等于它的最大值 $f(\frac{1}{3}) = \ln 3-\frac{1}{3}$。

总结

在高中导数题中，双变量问题出现的频率很高，这里的单变量代替法是很常用的方法，适用于给出了两个变量之间的倍数关系的情况，否则可能会因为单变量没有范围而放大或放小。

难点还是在求导，在肉眼看不出关系，又不好放缩时，多导几次是最好的选择，不过要注意导数之间 $>0$ 和单增的区别，别弄混淆了。

T6

题面

已知 $f(x) = 2\sin x-\ln (1+x)$，设 $f(x)$ 在 $(0,\pi)$ 内的极值点为 $\alpha$，零点为 $\beta$，比较 $2\alpha$ 和 $\beta$ 的大小。

解析

初步分析

先看看这个函数长什么样，之后再确定怎么做。

尝试解题

$f^{‘}(x) = 2\cos x-\frac{1}{1+x}$

显然在 $x\in (\frac{\pi}{2},\pi)$ 时，$f^{‘}(x)<0$，但是 $x\in (0,\frac{\pi}{2})$ 时不好分析，直接求二阶导：

$f^{‘’}(x) = -2\sin x+\frac{1}{(1+x)^2}$

可以看出，它在 $(0,\frac{\pi}{2})$ 上单减。

$f^{‘’}(0) = 1>0,f^{‘’}(\frac{\pi}{2}) = -2+\frac{1}{(1+\frac{\pi}{2})^2}<-2+1<0$

所以，一定存在 $x_0$，使 $f^{‘’}(x_0) = 0$。

于是，$f^{‘}(x)$ 在 $(0,x_0)$ 上单增，在 $(x_0,\frac{\pi}{2})$ 上单减。

$f^{‘}(0) = 2-1>0,f^{‘}(\frac{\pi}{2}) = -\frac{1}{1+\frac{\pi}{2}}<0$，所以在 $(0,x_0)$ 上都大于 $0$，之后减到小于 $0$，那么必然有一个点为 $0$，这既是极值点。

我们确定了 $\alpha \in (0,\frac{\pi}{2})$。

$f(0) = 0,f(\frac{\pi}{2}) = 2-\ln(1+\frac{\pi}{2})>2-\frac{\pi}{2}>0$

（这里用了 $\ln x<x-1$）

于是，$f(x)$ 在 $(0,\frac{\pi}{2})$ 上都大于 $0$。

$f(\pi) = -\ln(1+\pi)<0$，又因为 $f(x)$ 在 $(\frac{\pi}{2},\pi)$ 上单调递减，所以 $f(x)$ 在 $(\frac{\pi}{2},\pi)$ 上有一个零点。

再次分析

求 $2\alpha$ 和 $\beta$ 的大小，其实就是比较 $f(2\alpha)$ 和 $0$ 的大小，之后根据 $f(x)$ 在 $(\alpha,\pi)$ 上单调递减的性质即可推出大小关系。

继续解题

根据极值点，我们可以知道下面的等式：

$2\cos \alpha-\frac{1}{1+\alpha} = 0$

再没有其它的条件了，开始算 $f(2\alpha)$。

$f(2\alpha) = 2\sin 2\alpha-\ln(1+2\alpha) = 4\sin \alpha \cos\ \alpha-\ln(1+2\alpha)$

正好把条件用上：

$f(2\alpha) = \frac{1}{1+\alpha}2\sin \alpha-\ln(1+2\alpha)$

接下来只能求导了，不过这里有一个技巧，把分式先提出来，因为它和大小无关，所以可以忽略它，接下来的式子就没有分式了，好导一些。

$f(2\alpha) = \frac{1}{1+\alpha}(2\sin \alpha-(1+\alpha)\ln(1+2\alpha))$

$f^{‘}(2\alpha) = 2\cos\alpha-\ln(1+2\alpha)-2\frac{\alpha+1}{2\alpha+1} = 2\cos\alpha-\ln(1+2\alpha)-1-\frac{1}{2\alpha+1}$

$f^{‘’}(2\alpha) = -2\sin\alpha-\frac{2}{1+2\alpha}+\frac{2}{(1+2\alpha)^2} = 2(-\sin \alpha-\frac{1}{1+2\alpha}+\frac{1}{(1+2\alpha)^2})$

可以发现，后面的东西很类似于 $x^2-x$，然后这玩意在 $(0,1)$ 是恒小于 $0$ 的。

可是，我们的 $\alpha \in (0,\frac{\pi}{2})$，所以 $0<\frac{1}{1+2\alpha}<1$，这不正好吗？

前面的 $-\sin \alpha$ 显然小于 $0$，所以整个二阶导小于 $0$，也就是说一阶导单调递减。

$f^{‘}(0) = 2-0-1-1 = 0$，所以整个一阶导也小于 $0$，也就是说 $f(2\alpha)$ 单调递减。

$f(0) = 1(0-0) = 0$，那不就恒小于 $0$ 了吗？

于是，$f(2\alpha)<0 = f(\beta)$，所以 $2\alpha>\beta$。

总结

遇到这种题不要怕，函数复杂的就先把函数的大概走势看清楚。

导数题，就怕不求导，别看二阶导多么复杂的样子，实际上几阶导不都是导数吗，看不出来的再求导就完了。

最后就是基本函数也要敏感一点，免得求导浪费时间。

导数题，一切都为求导服务，所有的计算都是为了更清晰的看出函数的走势。

T7

题面

$f(x) = e^x-x$，当 $t>1$ 时，有 $\frac{1}{f^{‘}(\ln t)}>\frac{\lambda}{f^{‘}(-\ln t)}+\frac{\lambda+1}{\ln t}$，求正实数 $\lambda$ 的取值范围。

解析

初步分析

所有的条件都在题目中，直接开做就完了。

尝试解答

$f^{‘}(x) = e^x-1$

$\frac{1}{t-1}>\frac{\lambda}{\frac{1}{t}-1}+\frac{\lambda+1}{\ln t}$

化简，得 $\lambda>\frac{-\ln t+t-1}{t\ln t-t+1} = g(t)$

$g^{‘}(t) = \frac{\ln^2t-t-\frac{1}{t}+2}{(t\ln t-t+1)^2} = \ln^2t-t-\frac{1}{t}+2$

$g^{‘’}(t) = 2\ln t\frac{1}{t}-1+\frac{1}{t^2} = 2t\ln t-t^2+1$

$g^{‘’’}(t) = 2\ln t+2-2t$

由于 $2\ln t<2t-2$，所以 $g^{‘’’}(t)<0$，$g^{‘’}(t)$ 单减。

$g^{‘’}(1) = 0$，所以 $g^{‘}(t)$ 单减。

$g^{‘}(1) = 0$，所以 $g(t)$ 单减。

于是，我们只要求出 $g(1)$，即可知道 $\lambda$ 的最小值了。

但是，$g(t)$ 在 $t = 1$ 时分子分母都是 $0$，于是可以开洛。

$\lim_{t\rightarrow 1}\frac{-\ln t+t-1}{t\ln t-t+1} = \lim_{t\rightarrow 1}\frac{-\frac{1}{t}+1}{\ln t+1-1} = \lim_{t\rightarrow 1}\frac{\frac{1}{t^2}}{\frac{1}{t}} = 1$

总结

小清新导数题，只需要求出三阶导，即可快速判断单调性，再合理使用洛必达法则，即可快速出结果。

T8

题面

对于正实数 $a$ 有 $f(x) = e^{x+1}-a\ln x-\ln\frac{a^a}{e^a}\geq 0$ 在定义域内恒成立，求 $a$ 的取值范围。

解析

初步判断

简洁的题面，开导就完事了。

尝试解答

$f(x) = e^{x+1}-a\ln x-a\ln a+a$

$f^{‘}(x) = e^{x+1}-\frac{a}{x}$

容易发现导函数是单调递增的，所以只存在一个点 $x_0$，使得 $e^{x_0+1} = \frac{a}{x_0}$。

在 $(0,x_0)$ 上，$f^{‘}(x)<0$；在 $(x_0,+\infty)$ 上，$f^{‘}(x)>0$。

所以，$f(x)$ 在 $x_0$ 处取得最小值。

$f(x_0) = \frac{a}{x_0}-a\ln a+ax_0+a-a\ln a+a = \frac{a}{x_0}-2a\ln a+ax_0+2a$

$\frac{1}{x_0}-2\ln a+x_0+2\geq 0$

$2\ln a\leq x_0+\frac{1}{x_0}+2$

右边的用均值不等式，可以看出最小值为 $4$，此时 $x_0 = 1$。于是左边的最大值就是 $4$，所以 $2\ln a\leq 4$，$a\in (0,e^2]$。
