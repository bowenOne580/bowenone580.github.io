1988D

Link

考点：复杂度观察，朴素做法优化。

原先考虑的是黑白染色，之后发现不太行。虽然说理论上都可以两次删完（奇偶层），但不一定最优。

考虑这样的情况，一条 $4$ 个节点的链，链的首尾都是 $10000$，中间都是 $1$，显然先删首尾，再删两次中间，共三次最优。

dp 并不限制染色的数量，染色数量最大可以为 $n$，但是复杂度过高，为 $O(n^2)$。

现在有一棵染完色的树，假设颜色从 $1$ 到 $k$，那么这个染成 $k$ 的节点，和它相连的所有点，至少包含 $1$ 到 $k-1$。

定义 $f(i)$ 表示最大值为 $i$ 所需要的最少节点数量，从上面，至少有 $f(k)\geq 1+\sum_{i=1}^{k-1} f(i)$。手推一下，即可发现 $f(k)\geq 2^{k-1}$。

于是，染色数最大就是 $\log n$ 级别的了。复杂度成功降成 $O(n\log n)$。

所以，有时不要放弃朴素做法，多观察，可能稍加优化就过了。

1990D

Link

考点：观察

一行最多被操作两次。如果操作三次及以上，显然最多也只会影响上下三行，不如用一行染白的操作搞三次。

不好确定贪心策略，大概率是 dp。

用正方形删，要么删前面两个，要么删前面四个，否则肯定要删三次以上。

dp 转移就四种，要么不管或全删，要么删前面两个，要么删后面两个，要么删四个。

1994D

Link

考点：鸽巢原理

通过观察样例，发现根本没有 NO，一般有 NO 的题绝对会给个例子启发思路，于是考虑一定有解。

连边的顺序没有要求，但每次都要减少一个连通块，所以考虑先从限制大的开始连，即倒序连接。

之后发现，缩点后每次有 $x+1$ 个点，但只需要 $\mod x$ 即可连边，也就是说，肯定有两个点，它们 $\mod x$ 的值相同，相减后一定能被 $x$ 整除，可以连边，于是可以保证有解。

重点在于倒序思考。（我还以为要排序啥的）

2003D2

Link

考点：DP，建图

建议先看 D1，比较简单，每一个数列进行操作时，根据当前 $x$ 的不同，会得到两个不同的值，记为 $a,b$。

显然，当 $x = a$ 时，会得到 $b$，$x \neq a$ 时，会得到 $a$。

此题中，每个数列只能用一次，所以不能无限套娃。如果要得到最大值，需要在一个数列操作后，用另一个数列再次操作，考虑从每个数列的 $a$ 向 $b$ 连边，如果当前在某个点，显然可以通过数列操作去所有出边，到达最大的点即为所求。

问题在于，我们最开始可以在哪些点。显然不能从 $b$ 出发，只能从 $a$ 走，但如果最初的 $x$ 值不在 $a$ 里面，我们就需要先对一个数列操作得到 $a$，之后这个数列就用不了了。这等效于在图中的任意节点，删掉某一条出边，所能达到的最大节点。这是一个经典的 DP 问题，我选择从所有 $0$ 入度的点深搜维护 DP 数组。

如果初始的 $x$ 在 $a$ 或 $b$ 里面，就可以从所有出边走了，可以在上面的 DP 中顺便维护。

2007D

Link

考点：推理。

先简化题目中的定义：一个串中 $10$ 和 $01$ 的数目之差。朴素的解法是 $O(n)$ 的，手模一下即可发现，结果和中间的部分没有关系，如果将其中一个 $0$ 变成 $1$，对结果是没有影响的（长度为 $3$ 的串满足，扩展至长度为 $n$）。

所以，题中的 score 就是所有首尾不相等的串的数量。

于是，最为重要的就是根和叶子节点。

如果根是确定的，显然 Iris 需要染成和根不同的颜色，Dora 需要染成相同的颜色，最终结果显然。

如果根不确定，一种策略是看叶子节点中 $0$ 和 $1$ 的多少，由于 Iris 是先手，所以她一定会把根染成其中少的一个。

如果叶子节点中 $0$ 和 $1$ 数量相同，我们按照刚才的策略，相当于任选一个颜色，这时 Dora 一定会多染一个节点。扩展一下，相当于先染根的都会少染一个节点，必亏。所以我们需要看能否让对方先染根。

这时，还剩一些未确定的非根非叶子节点，先把它们染了，再看当前行动一方该如何选择。

如果她染根，就会亏一个节点，且之后的策略唯一。

如果她染叶子，当叶子是偶数个时没啥影响，当叶子有奇数个，她会染最后一个叶子，这时轮到对方染根，还是亏一个，不如染根。

这道题中最重要的是观察性质和分类讨论，策略反而更显然一点。
