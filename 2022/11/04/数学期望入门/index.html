数学期望入门

基本概念

我们把一个随机试验的结果称为样本点，这些结果组成样本空间。

对于样本空间内的一个随机事件 $A$，定义 $P(A)$ 为 $A$ 发生的概率。

假设一个随机变量 $X$ 的取值有 $n$ 个，分别为 $x_1,x_2\cdots x_n$，那么定义随机变量 $X$ 的数学期望为 $E(X) = \sum_{i=1}^np_i\cdot x_i$，也就是出现的概率乘上取值。

下面很重要！

$E(aX+bY) = a\cdot E(X)+b\cdot E(Y)$

证明：

  1. $E(X+Y) = \sum_{i=1}^np_ix_i+\sum_{i=1}^mq_iy_i = E(X)+E(Y)$
  2. $E(aX) = a\sum_{i=1}^np_ix_i = aE(x)$

当然，可以加的前提是样本空间相同。比如你掷骰子，两个骰子面数不同，那么最终的期望显然不能简单的相加，因为在总样本空间内，相对于两个骰子分别的样本空间，概率的变化不是相同的。

例1

P6154 游走

先把期望写出来：

$\sum p_ix_i$

发现这里的 $p_i$ 都是一样的，提出来。

$p\sum x_i$

于是就变为求一个有向无环图的路径条数和路径长度和。

进行拓扑排序，同时 dp。

设 $f[i]$ 为 $i$ 点的路径数量，$g[i]$ 为 $i$ 点的路径长度和。

每次转移时，$f[j]+=f[i]$，$g[j]+=g[i]+f[i]$。

注意路径的起点和终点可能相同，需要把 $f[i]$ 的初值设为 $1$。

例2

216. Rainbow的信号

由于概率只和段本身有关系，所以可以先粗略的算一下每个段被选择的概率。

如果段的长度为 $1$，那么被选择的概率为 $\frac{1}{n^2}$，由于$l>r$ 时，可以交换，所以如果段的长度大于 $1$，那么被选择的概率为 $\frac{2}{n^2}$。

看到这种只有位运算的，一般都是把每一位拆开算，这样可以少很多限制。

再具体看第 $k$ 位的情况。

假设某一位上值为 $1$，那么它对期望的贡献为 $\frac{1}{n^2}\cdot 2^k$。

再考虑长度 $\geq 2$ 的段的贡献。

枚举左端点，假设为 $i$。

and 和，在下一个 $0$ 之前都是 $2^k$，即记 $i$ 处往后第一个 $0$ 的位置为 $j$，那么对答案的贡献就是 $\frac{2\cdot (j-i)}{n^2}\cdot 2^k$。

or 和，在下一个 $1$ 之后都是 $2^k$，即记 $i$ 处往后第一个 $1$ 的位置为 $j$，那么对答案的贡献就是 $\frac{2\cdot (n-j+1)}{n^2}\cdot 2^k$。

xor 和，遇到 $0$ 不变，遇到 $1$ 取反，即遇见第一个 $1$ 和第二个 $1$ 之间是 $1$，第三个 $1$ 和第四个 $1$ 之间是 $1$。

显然，这三个所需的信息都可以预处理出来，于是就是 $O(n)$ 的，带上 $31$ 大常数。

例3

优惠券 Coupons

这道题需要用到一个结论：概率为 $p$ 的事件期望 $\frac{1}{p}$ 次发生。

假设现在有 $i$ 张牌，要得到新的牌，则概率为 $\frac{n-i}{n}$，根据上面的结论，这个事件期望在 $\frac{n}{n-i}$ 次后发生。

我们设 $X$ 表示所有从 $0$ 到 $1$，从 $1$ 到 $2$，这样的事件的集合。

则根据期望的线性相加，$E(X) = \sum_{i=0}^{n-1}\frac{n}{n-i}$

把 $n$ 提出来，里面的就是 $E(X) = n\sum_{i=0}^{n-1}\frac{1}{n-i}$，就是调和级数。

现在就可以算了。

例4

高妙题。

P1654 OSU!

（话说 osu! 真的好难啊，我连2星的谱都打不过）

看到这种每次增加的只和当前位有关的，一般都是 dp。

依次考虑每一位，设 $ans[i]$ 表示前 $i$ 位的答案，显然只有当第 $i$ 位的操作成功时才会对 $ans[i]$ 产生贡献。

假设我们已经求出了 $ans[i]$，现在要求 $ans[i+1]$。怎么求？

显然，在第 $i$ 位时，长度从 $0$ 到 $i$ 都有可能，假设对于每个长度 $l$，它出现的概率为 $p_l$，且第 $i+1$ 位为 $1$ 的概率为 $P$，则可以写出 $ans$ 的转移方程：

$ans[i+1] = ans[i]+P\cdot \sum_{l=0}^i p_l(3l^2+3l+1)$

解释一下后面那一块：

显然，能对答案产生贡献的只能是 $i+1$ 位为 $1$，也就是说后面的贡献产生的概率为 $P$，故要乘 $P$。

而这一位是 $1$，对所有 $l$ 来说，从 $l^3$ 变成了 $l^3+3l^2+3l+1$，但是我们只算加上的贡献，这是因为，这两个东西实际上是互斥的，也就是说，你只能选择其中的一个，所以还需要把选择原来的贡献，即 $ans[i]$ 加上去。

发现后面的期望不好求，而且里面还是一个多项式，考虑展开：

$3\sum_{l=0}^i p_l\cdot l^2+3\sum_{l=0}^ip_l\cdot l+\sum_{l=0}^i p_l$

设 $E(A_i) = \sum_{l=0}^i p_l$，$E(B_i) = \sum_{l=0}^i p_l\cdot l$，$E(C_i) = \sum_{l=0}^i p_l\cdot l^2$。

先看第一个，显然，在第 $i$ 位，所有长度从 $0$ 到 $i$ 的序列构成了整个样本空间，于是 $E(A_i) = 1$。

对于第二个，考虑第 $i$ 位的可能：

  1. 为 $0$，则以 $i$ 结尾的全 $1$ 段的长度只能为 $0$，$E(B_i) = (1-P)\cdot (p_0\cdot 0) = 0$。
  2. 为 $1$，则以 $i$ 结尾的全 $1$ 段的长度至少为 $1$，至多为 $i$，而这些长度对应发生的概率就等于 $i-1$ 时的 $0$ 到 $i-1$ 的长度发生的概率，所以可以通过 $E(B_{i-1})$ 转移，$E(B_i) = P\cdot (\sum_{i=0}^{i-1}(p_l\cdot (l+1))) = P\cdot (E(B_{i-1})+E(A_{i-1})) = P\cdot (E(B_{i-1})+1)$

再看第三个，考虑第 $i$ 位的可能：

  1. 为 $0$，则贡献为 $0$。
  2. 为 $1$，则 $l$ 从 $1$ 到 $i$，发生概率对应 $E(C_{i-1})$，$E(C_i) = P\cdot (\sum_{l=0}^{i-1}(p_l\cdot (l^2+2l+1))) = P\cdot (E(C_{i-1})+2E(B_{i-1})+1)$

于是就做完了。

例5

P4550 收集邮票

如果期望次数为 $x$ 次，那么答案就是 $\frac{x^2+x}{2}$。

根据期望的线性相加，需要求出次数的期望和次数平方的期望。

设 $f[i]$ 表示已经有了 $i$ 张牌，期望再取多少次取到所有牌。

这里先不说怎么求这个东西，谈一下为什么要这样设。

有很多人（比如我）一开始会直接设 $f[i]$ 表示拿到 $i$ 张牌的期望次数，这固然是好求的，但是别忘了，我们后面还要求平方，而在求这个时，我们需要知道关于它的等式。

假设我们设 $f[i]$ 表示拿到 $i$ 张牌的期望次数，显然，这个次数只能从已经有了 $i$ 张，拿了一个旧的，和只有 $i-1$ 张，拿了一个新的贡献出来。但是，直接这样贡献就出问题了，因为我们说概率都是对于同一样本空间而言的，期望的线性相加也建立在此之上，但是，你贡献的时候，前面一种情况的概率 $\frac{i}{n}$，是对于“已经有 $i$ 张牌，下一张牌可能的情况”这一样本空间而言，可是后面一种情况的概率 $\frac{n-i+1}{n}$，是对于“已经有 $i-1$ 张牌，下一张牌可能的情况”这另一个样本空间而言的，既然样本空间不是相同的，那么就不符合期望的线性相加前提，所以你得到的等式，实际上是不成立的。

再来看一看设为还要多少次才能拿全的好处：

  1. 拿到旧牌，概率为 $\frac{i}{n}$，对期望的贡献为 $\frac{i}{n}(f[i]+1)$。
  2. 拿到新牌，概率为 $\frac{n-i}{n}$，对期望的贡献为 $\frac{n-i}{n}(f[i+1]+1)$。

你会发现，这样一来，概率就对应着同一个样本空间了，所以期望也就是对的了。

于是列出等式如下：

$f[i] = \frac{i}{n}(f[i]+1)+\frac{n-i}{n}(f[i+1]+1)$

化简：

$f[i] = f[i+1]+\frac{n}{n-i}$

你会发现，这和例3中依靠期望的性质给出的式子一样，所以这样做是对的。

为什么只能倒推不能正推呢（如果你是像某篇题解一样利用高等数学正推的大佬，当我没说）？究其原因，还是因为如果拿到新牌，必然是从后面一项贡献过来，所以只能先把后面的期望算出来，再算这里的。设计状态的时候可以想一些极端的情况：比如 $i = n$ 是我们倒推的第一项，说明这东西的期望应该是个定值，如果拿到 $n$ 张牌了，显然期望再拿的次数为 $0$，于是就设计出 dp 的状态了。

再说明一个简单一点的问题：为什么平方的期望不是期望的平方？

你可能在算期望的平方时，直接把期望拿来平方一遍，但这样是不对的，因为期望本质上是概率乘上数值，而我们要求的期望的平方，只是在数值上面平方，而概率不变，所以这两者不等价。

于是，设 $g[i]$ 表示拿到 $i$ 张牌，还要拿次数的平方的期望，可以列出等式如下：

$g[i] = \frac{i}{n}(g[i]+2f[i]+1)+\frac{n-i}{n}(g[i+1]+2f[i+1]+1)$

化简：

$g[i] = \frac{i}{n-i}(2f[i]+1)+g[i+1]+2f[i+1]+1$

倒推转移即可。

例6

P1850 [NOIP2016 提高组] 换教室

这道题相对于前面几道来说，简单一点。

首先预处理出两点间最短路。

之后考虑 dp，设 $f[i][j][0/1]$ 表示前 $i$ 个时间段，用了 $j$ 次申请，这一次是否申请。

转移的时候注意，如果是从 $1$ 转移过来，那么要计算两个期望，一个是成功申请的，一个是申请失败的，因为每次申请成功都是有概率的，如果这次申请了，也要把这次成功和失败的期望分别计算，这一点想清楚了，这道题也就不难了。

例7

P4316 绿豆蛙的归宿

怎么越来越简单了啊，看来我的编排水平不太行啊。

根据前面的经验，我们设 $f[i]$ 表示走到 $i$ 期望还要走多远才能走到终点，这样可以保证概率都是对于 $i$ 而言的。

显然 $f[n] = 0$。
