再谈线性筛

筛质数

P3383 【模板】线性筛素数

线性筛之所以快，是因为它遵循一条重要的原则：每个合数都被它最小的质因子乘上一个数筛掉。

先看循环中的第一个 $if$，$vis$ 数组判断 $i$ 是否被筛过，如果被筛，就记录着筛掉它的最小质因子，$pri$ 数组是存储所有筛出来的质数的。

之后，看到下面的 $for$，里面枚举了所有的质数 $pri[j]$，第一个 $if$ 里面判断的分别是是否被最小的质数筛和是否超出了筛的范围，如果是，就 $break$ 掉。

后面的一行，相当于现在 $pri[j]$ 比 $i$ 中的最小质因子都要小，所以乘起来，乘积的最小质因子就是 $pri[j]$ 了。

很好理解是吗？我们来看一些拓展。

筛欧拉函数

欧拉函数的计算式是 $\varphi(x) = x\prod_{i=1}^{k}(1-\frac{1}{p_i})$，其中 $p_i$ 是 $x$ 的质因子，$k$ 是质因子的个数。

怎么得到的可以看约数，这里不再赘述。

现在，我们在线性筛的框架内做这个事。

显然，$\varphi(p) = p-1$，因为 $p$ 是质数，所以质因子只有它一个，代入上式可得。

考虑两个互质的数相乘，由于它们的质因子都不共享，所以后面的 $\prod$ 里面的东西不重叠，它们的乘积正好就是前面的相乘，所以当 $p,q$ 互质时，$\varphi(p\cdot q) = \varphi(p)\cdot \varphi(q)$。

如果它们不互质，情况就有点复杂了，但是，在线性筛中，只会是一个质数和一个整数相乘，我们只用判断这个整数中是否含有这个质数即可，这可以通过取模操作实现。

  1. 不含有，则这两个数互质，按照上面的结论，$\varphi(pq) = \varphi(p)\cdot \varphi(q)$。
  2. 含有，我们写出乘积的欧拉函数计算式，发现它 $\prod$ 后面的东西和 $\varphi(q)$ 完全相同，只是前面多了一个 $p$，于是 $\varphi(pq) = p\cdot \varphi(q)$。

筛莫比乌斯函数

莫比乌斯函数的形式化定义：

$n = \prod_{i=1}^k p_i^{c_i}$，其中 $p_i$ 是质因子，$c_i$ 是该质因子的指数，如果 $c_i$ 全部 $\leq 1$，则 $\mu(n) = (-1)^k$，否则 $\mu(n) = 0$。

考虑线性筛中 $p$ 和 $q$ 相乘，其中 $p$ 是质数。

如果 $p,q$ 互质，那么显然 $p$ 这个质因子没有在 $q$ 中出现，所以 $p\cdot q$ 中它的指数一定是 $1$，相当于让 $k$ 增加了 $1$，分讨一下，可以发现 $\mu(pq) = -\mu(q)$。

如果 $q$ 是 $p$ 的倍数，那么显然 $pq$ 中 $p$ 的指数大于 $1$，应该赋值为 $0$。

筛约数个数

根据唯一分解定理，$n = \prod_{i=1}^kp_i^{c_i}$，则约数个数为 $\prod_{i=1}^k(c_i+1)$。

设 $d[i]$ 表示 $i$ 的约数个数，$cnt[i]$ 表示 $i$ 的最小质因数出现次数。

显然，对于所有质数 $i$，$d[i] = 2,cnt[i] = 1$。

再次考虑线性筛中的 $p,q$。

  1. $p,q$ 互质，则 $p$ 这个质因数在 $pq$ 中出现的次数一定是 $1$，所以 $d[pq] = 2\cdot d[q],num[pq] = 1$。
  2. $q$ 是 $p$ 的倍数，则 $pq$ 中，$p$ 出现的次数为 $num[q]+1$，所以 $d[pq] = d[q]/(num[q]+1)\cdot (num[q]+2),num[pq] = num[q]+1$。

筛约数和

先给出约数和的计算式：

$d[x] = \sum_{i=1}^k(1+p_i+p_i^2+\cdots +p_i^{c_i})$

维护一个 $num[x]$ 表示 $x$ 的最小质因数和它的指数组成的 $d$。

还是考虑 $p,q$。

  1. $p,q$ 互质，则 $d[pq] = d[p]\cdot d[q]$，因为上面的质因数没有重叠，都对答案产生了贡献，又由于 $p$ 变成了 $pq$ 的最小质因数，所以 $num[pq] = p+1$。
  2. $q$ 是 $p$ 的倍数，则 $d[pq] = d[q]/num[q]\cdot (num[q]\cdot p+1),num[pq] = num[q]\cdot p+1$。
