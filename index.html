CH0x5 总结

DP 概念

DP 分为三步：阶段、状态、决策。

阶段，指计算时的顺序性，总是用上一个阶段的结果推导下一个阶段的结果，这样的转移顺序具有无后效性的特点。

状态，指一个阶段中，每个能够转移到的点。

决策，指转移状态的选取，即一个点被哪些点转移，或者一个点能够转移到哪些点。

浅谈反悔贪心

发表于 2025-07-14 Valine：

浅谈反悔贪心

例1

P3620 [APIO/CTSC2007] 数据备份 - 洛谷

楼之间连线肯定不会重叠，因为总可以交换使得答案更小。因此，选择相邻的 $k$ 对楼，且左右端点不重合一定最优。

朴素的想法是求出每两栋楼之间的距离，之后将这些数放到堆里面，取 $k$ 个最小的。

但是，两个楼被取走之后，它旁边的两个间隔也不能选了，有时选择两边的两个更优，于是贪心就出错了。

这就是反悔贪心的典型问题：

有 $n$ 个数 $a_1,a_2\cdots,a_n$，现在要选择 $k$ 个不相邻的数，使得总和最小/最大。

FFT 入门

多项式乘法

整数乘法，通常用长乘法（竖式乘法）进行运算，也就是将一个乘数的每一位和另一个乘数的每一位相乘，之后放到对应的位置上，最后加和。

多项式乘法，就是两个多项式相乘，其中 $P(x) = p_0+p_1 x+\cdots+p_{n-1}x^{n-1}$，$Q(x) = q_0+q_1 x+\cdots +q_{n-1}x^{n-1}$，要求 $P(x)\cdot Q(x)$ 里每一项的系数，显然有 $n-1+m-1+1 = n+m-1$ 项。

可以用类似整数乘法的方式，让系数两两相乘，加到对应的位置上，时间复杂度为 $O(n^2)$。

Color a Tree 题解

题面

按照一定顺序染色一个树上的所有点，第 $i$ 个被染色的权值为 $a$ 的点对答案的贡献为 $i\cdot a$。

思路

比较难想的贪心题，很容易想到最大值的特殊性，但是很难找到一个正确的贪心策略。

最大的点对答案的贡献影响最大，考虑选择最大点的时机。

但是，不是很好在一开始就确定最大点的精确时机，不过可以确定的是，最大点一定在它的父亲之后被染色。
